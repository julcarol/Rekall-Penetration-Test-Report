# Screenshots of Exploited Evidence

This folder contains selected screenshots demonstrating verified vulnerabilities
and successful exploitation during the Rekall Corporation penetration test.

Each image corresponds to a documented finding in the full assessment report.

## Included Evidence

- **Reflected XSS** – JavaScript execution via unsanitized input
- **Stored XSS** – Persistent script execution on page load
- **SQL Injection** – Authentication bypass via crafted payloads
- **Local File Inclusion (LFI)** – Access to sensitive system files
- **Command Injection** – Arbitrary command execution on the server
- **Directory Traversal** – Unauthorized file access outside intended paths
- **Apache Tomcat RCE** – Remote shell access via Metasploit
- **SLMail Buffer Overflow** – Remote Code Execution on Windows host
- **FTP Enumeration** – Anonymous access to sensitive files
- **Credential Dumping (Kiwi)** – Extraction of NTLM hashes post-exploitation
- **Network Reconnaissance** – Host and service discovery via Nmap

All sensitive data has been redacted where appropriate.
